User stories:

As a user, I would like to be able to create, remove, and delete appliances.
As a user, I would like to be able to set the yearly hour usage of appliance
As a user, I would like to be able to set the priority
As a user, I would like to be able to connect their car's data into the emission tracker
As a user, I would like to be able to view their yearly emissions
(Extra) As a user, I would like to be able to view a strategy, based on the priorities of user
(Extra) As a user, I would like to be able to add cost-viewing capabilities to the tracker
